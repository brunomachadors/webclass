Curso Web: Introdução

Aula 1: Elementos Básicos de HTML

Vamos explorar os principais elementos do HTML! Você verá como diferentes tipos de tags funcionam e para que servem.

Títulos

Os títulos são definidos pelas tags H1 até H6.
Quanto menor o número, maior o tamanho do título. 
<h1>H1: Mundial do Grêmio de 1983</h1>
<h2>H2: Mundial do São Paulo de 1993</h2>
<h3>H3: A Libertadores do Flamengo em 2019</h3>
<h4>H4: O Brasileirão do Palmeiras de 2018</h4>
<h5>H5: A Copa do Brasil do Cruzeiro em 2000</h5>
<h6>H6: O Mundial do Corinthians de 2000</h6>

H1: Mundial do Grêmio de 1983

H2: Mundial do São Paulo de 1993

H3: A Libertadores do Flamengo em 2019

H4: O Brasileirão do Palmeiras de 2018

H5: A Copa do Brasil do Cruzeiro em 2000
H6: O Mundial do Corinthians de 2000

Parágrafos

Parágrafos são usados para textos mais longos e são criados com a tag P. 
      <p>"Não concordo, nem discordo, muito pelo contrário."</p>

Não concordo, nem discordo, muito pelo contrário.

Textos com Estilo

Você pode estilizar o texto com tags específicas, como negrito Strong, itálico EM, sublinhado U e riscado S. 
<p><strong>Negrito:</strong> Este texto está usando a tag <strong>.</p>
<p><em>Itálico:</em> Este texto está usando a tag <em>.</p>
<p><u>Sublinhado:</u> Este texto está usando a tag <u>.</p>
<p><s>Riscado:</s> Este texto está usando a tag <s>.</p>

Negrito: Este texto está usando a tag <strong>.

Itálico: Este texto está usando a tag <em>.

Sublinhado: Este texto está usando a tag <u>.

Riscado: Este texto está usando a tag <s>.

Listas

Listas são usadas para agrupar itens relacionados. Existem dois tipos principais: listas ordenadas (numeradas) e listas não ordenadas (com marcadores). 
          
            <ol>
              <h4>Como ficar milhonário?</h4>
              <li><s>Estudar muito</s> (não funcionou)</li>
              <li><s>Trabalhar</s> (não funcionou)</li>
              <li><s>Investir</s> (não funcionou)</li>
              <li>Ser herdeiro</li>
              <li>Não fazer nada</li>
            </ol>
            
            <ul>
              <h4>Coisas para não fazer em um apocalipse zumbi</h4>
              <li>Um piquenique ao ar livre.</li>
              <li>Ir ao shopping.</li>
              <li>Pular o treino de cardio.</li>
              <li>Esquecer a munição em casa.</li>
            </ul>

Lista Ordenada

Como ficar milhonário?

  1. Estudar muito (não funcionou)
  2. Trabalhar (não funcionou)
  3. Investir (não funcionou)
  4. Ser herdeiro
  5. Não fazer nada

Lista Não Ordenada

Coisas para não fazer em um apocalipse zumbi

  • Um piquenique ao ar livre.
  • Ir ao shopping.
  • Pular o treino de cardio.
  • Esquecer a munição em casa.

Imagens

Imagens são adicionadas com a tag IMG. O atributo SRC define o caminho da imagem, e o ALT descreve a imagem. 
          <img src="https://res.cloudinary.com/dtglidvcw/image/upload/v1734797276/webaula/profile.jpg alt="Homem levemente embriagado"" />

Links

Links são criados com a tag A e podem apontar para outras páginas ou sites. Use o atributo HREF para definir o endereço do link. 
          <a href="https://brunomachadors.github.io/bugbuster/" target="_blank">Bug Buster</a>
Bug Buster

Tabelas

Tabelas são criadas com a tag TABLE, com linhas (TR) e colunas (TH e TD). 
      
      <table>
        <tr>
          <th>Pessoa</th>
          <th>Dias sem sair da dieta</th>
          <th>Tentativas de fugir da academia</th>
        </tr>
        <tr>
          <td>Ana</td>
          <td>3</td>
          <td>7</td>
        </tr>
        <tr>
          <td>Bruno</td>
          <td>0</td>
          <td>15</td>
        </tr>
        <tr>
          <td>Carlos</td>
          <td>10</td>
          <td>2</td>
        </tr>
      </table>
