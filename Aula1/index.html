Aula 1: Elementos Básicos de HTML

Vamos explorar os principais elementos do HTML! Você verá como diferentes tipos de tags funcionam e para que servem.

Títulos

Os títulos são definidos pelas tags <h1> até <h6>. Quanto menor o número, maior o tamanho do título. 
<h1>H1: Mundial do Grêmio de 1983</h1>
<h2>H2: Mundial do São Paulo de 1993</h2>
<h3>H3: A Libertadores do Flamengo em 2019</h3>
<h4>H4: O Brasileirão do Palmeiras de 2018</h4>
<h5>H5: A Copa do Brasil do Cruzeiro em 2000</h5>
<h6>H6: O Mundial do Corinthians de 2000</h6>

H1: Mundial do Grêmio de 1983

H2: Mundial do São Paulo de 1993

H3: A Libertadores do Flamengo em 2019

H4: O Brasileirão do Palmeiras de 2018

H5: A Copa do Brasil do Cruzeiro em 2000
H6: O Mundial do Corinthians de 2000

Parágrafos

Parágrafos são usados para textos mais longos e são criados com a tag <p>. 
<p>"Programar é como cozinhar: você começa com a receita, mas termina improvisando."</p>
<p>"Front-end é como mágica: você impressiona os outros, mas sabe que é tudo gambiarra."</p>
<p>"Debugging: o processo de remover o bug do código e, em seguida, adicionar mais dois no lugar."</p>
<p>"HTML sem CSS é como um bolo sem cobertura: funcional, mas sem graça."</p>

"Programar é como cozinhar: você começa com a receita, mas termina improvisando."

"Front-end é como mágica: você impressiona os outros, mas sabe que é tudo gambiarra."

"Debugging: o processo de remover o bug do código e, em seguida, adicionar mais dois no lugar."

"HTML sem CSS é como um bolo sem cobertura: funcional, mas sem graça."

Textos com Estilo

Você pode estilizar o texto com tags específicas, como negrito, itálico e sublinhado. 
<p><strong>Negrito:</strong> Este texto está usando a tag <strong>.</p>
<p><em>Itálico:</em> Este texto está usando a tag <em>.</p>
<p><u>Sublinhado:</u> Este texto está usando a tag <u>.</p>
<p><s>Riscado:</s> Este texto está usando a tag <s>.</p>

Negrito: Este texto está usando a tag <strong>.

Itálico: Este texto está usando a tag <em>.

Sublinhado: Este texto está usando a tag <u>.

Riscado: Este texto está usando a tag <s>.
